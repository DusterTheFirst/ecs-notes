## Turtle

```python
import turtle

turtle.color('red', 'yellow')
turtle.forward(10)
turtle.left(120)

turtle.setx(-100)
turtle.sety(100)

turtle.penup()
turtle.pencolor('red')
turtle.pendown()

turtle.done()

# x: -180 -> 180
# y: -200 -> 200
# Center is 0,0
```
